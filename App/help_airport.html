Help - Airport code

If you need the weather from a given airport (ex: LSGG for Geneva Cointrin), then enter the 4-character ICAO code and leave the country code empty.

If you need the weather from several airports, you have 3 options:

  • List
    Enter the ICAO codes, separated by the comma sign (ex: LFPO,EGLL,LSGG) for Paris Orly, London Heathrow and Geneva Cointrin
  • Flightpath
    Enter the ICAO codes of the departure, in-transit (optional) and arrival airports, separated by the semicolon sign (ex: LFPG;LFMN;LIRF) for any airport within the flightpath from Paris Charles de Gaulle to Rome Fiumicino via Nice Cote d'Azur
  • Wild card
    Enter the first letters of the ICAO codes (ex: LF for all French airports)
