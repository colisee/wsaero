Help - Output

The Output field allows you to show the results in different ways:

  • HTML
    This is the default choice. Weather data is formatted for your web browser.
  • RSS
    Weather data is formatted for any Really Simple Syndication-compatible software.
  • KML
    Weather data is returned with the KML format.
  • XML
    Weather data is returned with an XML format.
  • GEOJSON
    Weather data is returned with the GeoJSON format.
